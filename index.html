5-Day Curriculum Roadmap Table

This visual roadmap is designed to give parents and instructors a clear, glanceable overview of what the young innovators will build, learn, and showcase each day.

Day 01

The Spark & Electronics Foundation

Project Core

The Smart Night Light

  • Introduction to microcontrollers and breadboards.
  • Learning circuit mapping using resistors and LEDs.
  • Programming an LDR sensor to monitor real-time light thresholds.
Technical Pillar: Basic Circuitry & Code Logic
Parents' Corner Skill: Inventory Management
Day 02

Robotics & Actuators

Project Core

The Autonomous Crawler

  • Understanding DC motors, power delivery, and voltage regulation.
  • Wiring and controlling an L298N/L293D motor driver.
  • Assembling a 2WD/4WD robotic chassis from scratch.
Technical Pillar: Mechanical Assembly & Motor Control
Parents' Corner Skill: Workspace Etiquette
Day 03

Intelligent Navigation Systems

Project Core

The Smart Obstacle-Avoiding Car

  • Integrating ultrasonic sonar technology to measure physical distance.
  • Calibrating a servo motor to scan the surroundings dynamically.
  • Writing nested If-Else structures for real-time decision-making.
Technical Pillar: Autonomous Navigation & Critical Coding
Parents' Corner Skill: Failure Analysis & Debugging
Day 04

Internet of Things (IoT) & Automation

Project Core

IoT Smart Irrigation System

  • Connecting microcontrollers to global high-speed Wi-Fi networks.
  • Interfacing analog soil moisture sensors with electromagnetic relays.
  • Routing real-time environmental data to a cloud-based dashboard.
Technical Pillar: Cloud Systems & Automation Engineering
Parents' Corner Skill: Empathy-driven Social Impact Thinking
Day 05

Grand Innovation Challenge

Deliverable Core

The Great Autonomous Race & Tech Pitch

  • The Mini Hackathon: Tearing down and fixing code bugs in a time-sensitive environment.
  • Running fully autonomous cars through a complex obstacle track.
  • Conducting tech project presentations.
Technical Pillar: End-to-End System Integration
Parents' Corner Skill: English Public Speaking, Leadership, and Presentation

The Pro-Maker Discipline

At ZAN Tech, we don't just teach technology; we cultivate lifelong professional habits throughout the 5 days.

Inventory Management

Each student manages their own dedicated components toolbox, counting sensors and wires before and after every session to build high responsibility.

Workspace Etiquette

Following the 'Clean your bench before you leave' rule to instill workplace organization and professional discipline.

Time Management

Working against strict milestone deadlines to balance focus, precision, and speed under real-world scenarios.
